# LIRA UNIVERSITY PAST PAPER REPOSITORY (LU Academic Hub)

A comprehensive academic resource management system for Lira University, providing access to past papers, learning materials, and collaborative tools for students and faculty.

## Features

### Core Features
- 📚 **Past Paper Repository** - Upload, search, and download past examination papers
- 🔍 **Advanced Search** - Filter by faculty, department, course, year, semester, and more
- 👥 **User Management** - Student, Lecturer, Admin, and Super Admin roles
- 💬 **Discussion Forum** - Q&A, course discussions, and academic help
- 📝 **Learning Materials** - Lecture notes, assignments, projects, and tutorials
- 🔬 **Research Repository** - Final year projects, dissertations, and research papers
- 📊 **Analytics Dashboard** - Track uploads, downloads, and user activity
- 🔔 **Real-time Notifications** - Stay updated on new content and interactions
- 🌓 **Dark/Light Mode** - User preference for comfortable viewing
- 📱 **Mobile Responsive** - Access from any device

### Security Features
- 🔐 **Secure Authentication** - Password hashing, CSRF protection
- 🛡️ **XSS Prevention** - Input sanitization and output encoding
- 🚫 **SQL Injection Protection** - Prepared statements with PDO
- 📋 **Audit Logging** - Track all user activities
- 🔒 **Role-based Access Control** - Granular permissions per role

## Technology Stack

### Frontend
- HTML5, CSS3, JavaScript (Vanilla ES6)

### Backend
- PHP 8+
